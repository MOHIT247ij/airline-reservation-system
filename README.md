# ✈️ Airline Reservation System

A full-stack web application for managing airline reservations, 
built with React and Firebase.

## 🛠️ Tech Stack
- **Frontend:** React 19, Tailwind CSS, React Router DOM
- **Backend:** Firebase (Firestore Database)
- **Build Tool:** Vite
- **Data:** XML Import/Export support

## ✨ Features
- Airline dashboard with real-time data
- XML data import functionality
- Firebase Firestore integration
- Responsive UI with Tailwind CSS

## 🚀 How to Run Locally
```bash
git clone https://github.com/MOHIT247ij/airline-reservation-system.git
cd airline-reservation-system
npm install
npm run dev
```
